# Projet : Machine Learning - Classification

Ce projet illustre l'application de **techniques de Machine Learning supervisé** pour développer un **modèle de classification performant**.  
L’objectif est de créer un modèle capable de **prédire correctement la classe d’un échantillon** à partir de différentes variables.

## Objectif du Projet
- Développer un modèle de classification fiable  
- Comprendre les facteurs influençant les prédictions  
- Évaluer et optimiser les performances des modèles  

## Principales étapes d'un projet Machine Learning
1. **Récupération et vérification des données** : import des datasets et contrôle de leur qualité.  
2. **Exploration et préparation des données** : nettoyage, traitement des valeurs manquantes, encodage et normalisation.  
3. **Développement des modèles** : entraînement de différents algorithmes de classification (Logistic Regression, Random Forest, SVM, etc.).  
4. **Évaluation des modèles développés** : utilisation de métriques comme accuracy, F1-score, précision et rappel.  
5. **Recherche des meilleurs hyperparamètres** : optimisation des modèles via Grid Search ou Random Search pour améliorer les performances.

## Technologies Utilisées
- Python : Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn  
- Jupyter Notebook pour le développement et la documentation
